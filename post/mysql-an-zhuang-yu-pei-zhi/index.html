mysql 安装与配置

2020-06-17
  1. 下载免安装包并解压
  2. 环境变量添加 D:\Program Files\mysql-5.7\bin(mysql解压文件夹路径 as 『path』)
  3. 『path』下新建 my.ini 文件:
[mysqld]
basedir=D:\Program Files\mysql-5.7\
datadir=D:\Program Files\mysql-5.7\data\
port=3306
skip-grant-tables
  1. 『path』下执行 mysqld –install安装服务
  2. 『path』下新建data文件夹，启动mysql服务，再执行mysqld --initialize-insecure --user=mysql;
  3. 执行mysql –uroot –p 登录
  4. 运行update mysql.user set authentication_string=password('123456789') where user='root' and Host = 'localhost';更改root密码；flush privileges刷新权限
  5. 修改 my.ini文件删除最后一句skip-grant-tables并重启mysql
  6. 创建可远程连接用户
create user aowei identified by 'aowei';
grant all privileges on *.* to 'aowei'@'%'identified by 'aowei' with grant option;
flush privileges;
