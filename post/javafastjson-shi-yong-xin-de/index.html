（JAVA）fastjson使用心得

2018-08-10

  1. 在model类的指定字段上加上 @JSONField(name = "xxx") 能够在json转类对象时 对key为xxx的字段 转换为 进行了相应注解的model类字段。

  2. 在model类的指定字段上加上 @JSONField(serialize = false) 能够在json转类对象时 忽略进行了相应注解的model类字段。

  3. 创建PropertyFilter对象，如下：

     PropertyFilter filter = (object, key, value) -> {
 	//过滤处理
 };

 在其中进行过滤处理，再在对象转json数组时传入filter对象，能够对一些数据进行过滤处理
