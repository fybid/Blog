  1. ** Fatal: TaskCanceledException encountered.的解决方法 **

    解决方法：git config –global credential.helper manager

  2. ** Your local changes to the following files would be overwritten by checkout **

    解决方法：有文件未提交，可以选择提交或者清除未跟踪文件(git clean -n 和 git clean -f)
