记录一些Linux好用的命令

2020-08-13
  • IP/DNS等网络设置在 /etc/sysconfig/network-scripts
  • 注意 无法访问到某些域名的时候 可能是因为 /etc/sysconfig/network-scripts 下 当前使用的网卡的网络没设置DNS
  • 禁用/启用用户 sudo passwd -l/-u 用户名
  • 给用户管理员权限需要在 /etc/sudoers中修改
  • 编辑文本时 快速定位的方法 输入 '/要查找的字符'
  • tar -xvf 和 unzip 是不一样的
  • rm -rf 可以删除非空文件夹
  • cat log日志
  • scp src dst 复制
  • mv 移动文件兼重命名
  • systemctl 系统服务控制
  • jps -l 可以看到java进程和其pid
  • jstack -l pid > /path/filename
  • ln -s /mnt/f/javaProject/ ~/jproject 建立文件夹路径的快捷方式
  • grep "word" test.txt 从文本中查找关键字
  • ss -anptl 检查服务 端口 地址 进程id 等信息
  • timedatectl 时间控制
    set-time "yyyy-MM-dd HH24:mi:ss"
    set-timezone Asia/Shanghai
  • du -h --max-depth=1 查看当前一层文件夹大小
  • history 历史命令
  • top 任务管理器?
  • last 最近登陆成功的ip地址
  • lastb 最近没有登陆成功的ip地址
  • crontab -l 定时执行的程序
