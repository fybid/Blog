（SpringBoot）启动问题收集

· 2018-07-09 · # 框架

  1. Your ApplicationContext is unlikely to start due to a @ComponentScan of the default package.

    错误说明：你的java文件需要被放进一个包，启动文件需要在一个目录下

  2. Cannot determine embedded database driver class for database type NONE

    错误说明：配置文件中缺少数据源的相关配置

  3. Unsatisfied dependency expressed through field

    错误说明：代码规范问题，test中可能会将首字母自动识别成小写

  4. org.springframework.beans.factory.UnsatisfiedDependencyException：Error creating bean with name ‘xxx’...

    错误说明：bean 配置问题.. 可能是Repository中有找不到的变量(传参找不到也是可能的)..

  5. NoSuchMethodError: javax.persistence.Table.indexes()

    错误说明：发现是Bean的Table注解版本旧了，删除了lib让项目自行寻找新的去就可以了
