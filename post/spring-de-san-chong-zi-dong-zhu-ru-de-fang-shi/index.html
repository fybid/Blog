（Spring）三种自动注入的方式

2018-08-04

1.变量（filed）注入

@Autowired
UserDao userDao;

2.构造器注入

final
UserDao userDao;

@Autowired
public UserServiceImpl(UserDao userDao) {
    this.userDao = userDao;
}

3.set方法注入

private UserDao userDao;

@Autowired
public void setUserDao (UserDao userDao) {
    this.userDao = userDao;
}

编译器会在你使用第一种时提出警告并推荐使用第三种

大概是因为某些情况会报空指针异常的关系
