Nginx配置

2018-07-27

根据请求转发到不同服务器

配置文件修改如下：

http {
    server {
        listen       80;
        server_name  localhost;

        location /app_login {
                proxy_pass http://server1:port1/login;
                proxy_method GET;
        }

        location /sys_login {
                proxy_pass http://server2:port2/login;
                proxy_method POST;
        }
    }
}

访问location/app_login和location/sys_login就能看到效果了

均衡负载

upstream site {
    server 地址1:端口1;
    server 地址2:端口2;
}

server {
    listen       80;
    server_name  127.0.0.1;

    location / {
        proxy_pass http://site;
    }
}

好像还有不同的策略，具体可百度。
另外要注意session保持问题..有缓存或持久化session对象的设定之类的项目，可以忽视..

常用命令

重载配置：nginx -s reload
强制关闭：nginx -s stop
正常关闭：nginx -s quit
