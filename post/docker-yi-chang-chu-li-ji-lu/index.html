当docker容器出现"iptables failed: iptables --wait -t nat -A DOCKER..."报错时

解决方法来源于 kyrin28 的简书

pkill docker
iptables -t nat -F
ifconfig docker0 down
brctl delbr docker0
docker -d
systemctl restart docker
